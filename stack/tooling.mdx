Everything a teammate or agent gets by opening the repo. Rosters live here and in committed config, nothing ad-hoc on individual machines.

## Servers

Preconfigured in `.cursor/mcp.json`, Cursor lists them automatically after you open the repo. Approve the OAuth prompt in the browser once, done.

| Name | Description |
| --- | --- |
| [Mintlify](https://mintlify.com/docs) | Agents read and search this docs hub |
| [Linear](https://linear.app/docs/mcp) | Agents create, read, and update issues |

New servers, skills, and dependencies are agreed in Slack or Linear and land here via PR.

- `.specify/memory/constitution.md` matches the rules on the [architecture pages](/architecture/overview)
