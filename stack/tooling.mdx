Everything a teammate or agent gets by opening the repo. Rosters live here and in committed config, nothing ad-hoc on individual machines.

## Servers

Preconfigured in `.cursor/mcp.json`, Cursor lists them automatically after you open the repo. Approve the OAuth prompt in the browser once, done.

| Name | Description |
| --- | --- |
| Mintlify | Agents read and search this docs hub |
| Linear | Agents create, read, and update issues |

New servers, skills, and dependencies are agreed in Slack or Linear and land here via PR.

## Skills

Official vendor sources only, pinned via `skills-lock.json`.

| Name | Description |
| --- | --- |
| Mintlify | Components, frontmatter, and writing standards for this site |

Install: `npx skills add <source>`.

- First agent action on a missing spec: `/speckit.specify`
- `.specify/memory/constitution.md` matches the [guardrails](/concept/workflow#guardrails)
