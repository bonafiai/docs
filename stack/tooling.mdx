The agent and teammate workbench you get by opening the repo, versioned in committed config, nothing ad-hoc on individual machines. The product stack lives in [Overview](/stack/overview).

## Servers

Preconfigured in `.cursor/mcp.json`, Cursor lists them automatically after you open the repo. Approve the OAuth prompt in the browser once, done.

| Name | Description |
| --- | --- |
| [Mintlify](https://mintlify.com/docs) | Agents read and search this docs hub |
| [Linear](https://linear.app/docs/mcp) | Agents create, read, and update issues |

New servers and skills are agreed in Slack or Linear and land here via PR.

## Commands

The spec-kit [`/speckit.*` commands](https://github.github.io/spec-kit/quickstart.html) ship in the repo, wired for Cursor through the [`cursor-agent` integration](https://github.github.io/spec-kit/reference/integrations.html), so opening the repo puts them in chat with no per-machine install. The loop runs on them.

| Name | Description |
| --- | --- |
| `/speckit.constitution` | Write or update `.specify/memory/constitution.md` |
| `/speckit.specify` | Turn an issue into `spec.md` |
| `/speckit.clarify` | Resolve ambiguity before planning |
| `/speckit.plan` | Generate the implementation plan |
| `/speckit.checklist` | Quality-check the plan |
| `/speckit.tasks` | Break the plan into tasks |
| `/speckit.analyze` | Check spec, plan, and tasks agree |
| `/speckit.implement` | Execute the tasks |
| `/speckit.converge` | Confirm the feature is complete, queue any gaps |

One person runs `specify init . --integration cursor-agent` to wire them and `specify integration upgrade cursor-agent` after a spec-kit bump, both need [`uv`](https://docs.astral.sh/uv/). Everyone else just clones.

## Contract

The code repo's root `AGENTS.md` states:

- Agents file issues to the [Issues](/method/issues) bar
- No feature work without `specs/NNN-*/spec.md`
- First agent action on a missing spec: `/speckit.specify`
- `.specify/memory/constitution.md` matches the rules on the [architecture pages](/architecture/structure)
